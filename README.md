# Coffee Sales Dashboard

## Project Overview
This project is an interactive Coffee Sales Dashboard created in Microsoft Excel to analyze sales performance, customer behavior, and product trends. The dashboard provides insights through dynamic charts, Pivot Tables, and slicers for better business decision-making.

---

## Features
- Interactive sales dashboard
- Pivot Tables and Pivot Charts
- Dynamic slicers and filters
- Sales trend analysis
- Country-wise sales comparison
- Top customer analysis
- Product category insights

---

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization
- Dashboard Design

---

## Dashboard Insights
The dashboard helps analyze:
- Total sales over time
- Sales by country
- Top-performing customers
- Coffee roast type performance
- Product size analysis
- Loyalty card customer segmentation

---

## Project Workflow
1. Cleaned and organized the dataset
2. Created Pivot Tables for data summarization
3. Built Pivot Charts for visualization
4. Added slicers for interactive filtering
5. Designed and formatted the final dashboard
